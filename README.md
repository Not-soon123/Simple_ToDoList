# Simple To-Do List App

A simple **To-Do List web application** built using **HTML, CSS, and JavaScript**.
This project helped me practice **DOM manipulation, dynamic element creation, and event handling** in JavaScript.

## Features

* Add new tasks to the list
* Mark tasks as completed (toggle style)
* Delete tasks
* Add tasks using the **Enter key**
* Dynamic creation of list elements using JavaScript

## What I Practiced

In this project I focused on learning and practicing:

* **DOM Manipulation**
* Dynamically creating elements using `createElement()`
* Appending elements to the document
* **Toggling CSS classes** to change styles
* Handling **keyboard events** (`keydown` / `keyup`)
* Basic **event listeners** in JavaScript

## Technologies Used

* **HTML** – Structure of the application
* **CSS** – Styling the interface
* **JavaScript** – Logic and interactivity

## What I Learned

This project helped me better understand:

* How JavaScript interacts with the **DOM**
* How to dynamically create and update elements
* How to use **event listeners** for user interaction
* How to build small interactive web applications
